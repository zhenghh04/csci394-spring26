# Introduction to AI and Machine Learning

[Huihuo Zheng](https://www.alcf.anl.gov/about/people/huihuo-zheng), Argonne Leadership Computing Facility, <huihuo.zheng@anl.gov>

This module introduces the core ideas behind AI training using a simple
linear regression example. Students will learn:

- What AI training means (data-driven learning vs. rule-based programming)
- How large language models work at a high level
- How to fit a linear model using both an analytical formula and
  stochastic gradient descent (SGD)
- Key concepts: loss functions, learning rate, mini-batch training
