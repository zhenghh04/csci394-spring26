# CSCI 394 - High Performance Computing

This github repo contains example codes that is used in CSCI 394 - High Performance Computing class at Wheaton College

## What you will find here
- Small, focused examples that illustrate core HPC concepts.
- Reference implementations used in lectures and labs.
- Starter code for assignments and projects.

## Topics covered
- Parallel programming models (shared and distributed memory).
- Performance measurement and profiling.
- Data movement and memory hierarchy considerations.
- Scalability and speedup analysis.

## Examples
- [00_linux_env](00_linux_env) - Linux command line basics.
- [00_powershell_env](00_powershell_env) - PowerShell command line basics.
- [01_mpi_pi](01_mpi_pi) - MPI4PY Monte Carlo pi example + scaling.
- [02_pi_gpu](02_pi_gpu) - GPU/CPU pi example (PyTorch).
- [03_pi_torch_dist](03_pi_torch_dist) - Distributed PyTorch pi with `mpiexec`.

## Getting started
1. Clone the repository.
2. Review the folder for the week or topic you are working on.
3. Follow any local README files included with each example.
